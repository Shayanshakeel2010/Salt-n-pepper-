
# Salt-n-pepper

**Author:** Shayan

This portfolio website is built using **HTML, CSS, and Bootstrap**. It showcases modern web design techniques and responsive layouts.

## Features

* Add and display pictures.
* Apply attractive background designs using **Bootstrap** and custom CSS.
* Use CSS properties such as **hover effects**, transitions, and styling.
* Style paragraphs with custom CSS.
* Add shadow effects to headings for a modern appearance.
* Create a responsive and user-friendly interface that works on different screen sizes.
